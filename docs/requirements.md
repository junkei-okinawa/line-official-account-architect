# Line Official Account Architect

## Requirements

概要: LINE公式アカウントの構築を、LINEのチャット画面でAIとの対話を通じて行う。

大きく分けると**LINE公式アカウントのAPI**の操作と使って作成できる機能と**外部サービスとの連携、または自作機能**を組み合わせて作成できる機能に分けることができる。

**LINE公式アカウントのAPI**を使う昨日はMCPサーバーを構築することでAI向けのインターフェースを提供する方法を優先事項とする。

一部の機能にとどまるが、すでにLINEが公式でベータ版としてMCPサーバーを提供している。

- [line-bot-mcp-server](https://github.com/line/line-bot-mcp-server)
